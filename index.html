Инструкция по использованию

🎯 Основное назначение

Программа анализирует баскетбольные матчи в реальном времени и предоставляет рекомендации по ставкам на тоталы (ТБ/ТМ) с учетом множества факторов.

📱 Как пользоваться

1. Начало работы

  • Откройте программу в браузере
  • Выберите подходящую тему (светлая/темная) кнопкой в правом верхнем углу

2. Настройка параметров матча

Лига:

  • Выберите тип лиги (NBA, ФИБА, 3×3, 4×4, Кибербаскетбол)
  • Программа автоматически подстроит расчеты под выбранный формат

День недели:

  • Укажите день проведения матча
  • Программа учитывает статистику результативности по дням недели

Усталость команд:

  • Выберите уровень усталости команд:
    • Отдохнувшие (+2 дня отдыха)
    • Нормальный отдых (+1 день)
    • Back-to-back (2 игры подряд)
    • 3 игры за 4 дня

Пол команд:

  • Мужчины, женщины или смешанные команды

3. Загрузка данных

Способ 1: Скриншот (рекомендуется)

  • Нажмите "Выбрать файл" в области загрузки
  • Или перетащите скриншот матча в выделенную области
  • Программа автоматически распознает:
    • Счет по четвертям
    • Текущий период
    • Оставшееся время
    • Тотал и тип ставки
    • Фору

Способ 2: Ручной ввод

  • Заполните поля в разделе "Или введите данные вручную":
    • Счет (формат: 100:92)
    • Период (1-4)
    • Время (формат: 01:27)
    • Тотал (например: 237.5)
    • Тип тотала (ТБ/ТМ)
    • Фора (например: -6.5)

4. Просмотр результатов

Вкладка "Результаты"

  • Прогноз тотала на оставшуюся часть матча
  • Темп игры
  • Прогноз итогового счета
  • Вероятность овертайма
  • Рекомендация по ставке с указанием уверенности

Вкладка "Статистика"

  • Текущий темп игры
  • Сравнение со средним темпом лиги
  • Прогресс матча
  • Эффективность атаки
  • PPG команд

Вкладка "Прогноз по четвертям"

  • Прогноз очков во 2-й и 3-й четвертях
  • Рекомендации по ставкам на четверти
  • Учет лунной фазы и других факторов

Вкладка "Факторы влияния"

  • Детальный анализ влияния дня недели
  • Учет усталости команд
  • Влияние пола игроков
  • Лунный коэффициент
  • Суммарное влияние на результативность

🔍 Советы по использованию

Для лучшего распознавания скриншотов:

  • Используйте четкие скриншоты с хорошим контрастом
  • Убедитесь, что текст хорошо читаем
  • Желательно, чтобы на скриншоте были видны:
    • Счет по четвертям
    • Текущее время матча
    • Тотал и фора

При ручном вводе:

  • Используйте кастомную клавиатуру для ввода данных
  • Следите за правильным форматом:
    • Счет: 100:92
    • Время: 05:30
    • Тотал: 215.5
    • Фора: -4.5

📊 Как интерпретировать результаты

Рекомендации по ставкам:

  • 🟢 ЗЕЛЕНЫЙ блок - рекомендуемая ставка
  • Процент уверенности показывает надежность рекомендации
  • Учитывайте все факторы влияния перед принятием решения

Важные показатели:

  • Темп игры > 2.0 - высокая результативность
  • Темп игры < 1.5 - низкая результативность
  • Влияние факторов > +3% - повышенная результативность ожидается
  • Влияние факторов < -3% - пониженная результативность ожидается

⚠️ Важные примечания

  • Программа использует статистические модели и исторические данные
  • Все расчеты носят рекомендательный характер
  • Учитывайте текущую форму команд, травмы и другие факторы
  • Делайте ставки ответственно

🔄 Обновление данных

Программа автоматически:

  • Рассчитывает лунные фазы
  • Обновляет статистику по дням недели
  • Учитывает выбранные параметры лиги

Для обновления анализа просто измените любой параметр или загрузите новые данные.
